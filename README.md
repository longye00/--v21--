# 🌱 专业版 · 小白菜种子自动计数仪 v2.1

> 基于距离变换 + 分水岭算法的高精度种子计数 Web 应用  
> 支持粘连分离、手动校正、实时预览 — 纯前端，无需后端

---

## ✨ 功能特性

- **🌾 自动计数**：上传图片或拍摄，自动检测并计数小白菜种子
- **🧠 智能粘连分离**：采用距离变换 + 分水岭算法，精准分离粘连种子
- **🔧 可调参数**：灰度阈值、最小面积、分离强度、种子半径均可实时调节
- **🤖 自动阈值 (Otsu)**：一键计算最佳二值化阈值
- **👁 二值化/距离图预览**：可视化图像处理中间结果，便于调参
- **✏️ 手动校正**：添加遗漏种子、删除误检，计数更可靠
- **📏 多种选区模式**：多边形、矩形、全图自由选择计数区域
- **📸 摄像头支持**：直接拍摄照片进行计数
- **⌨️ 全键盘快捷操作**：Enter 计数、Backspace 撤销点、C 校正、P 预览等
- **🌈 图像预处理**：对比度/亮度调节，适应不同拍摄环境
- **📊 统计信息**：显示面积均值/最小/最大，自动计算发芽率
- **📱 移动端适配**：响应式设计，支持触屏操作、双指缩放

## 📖 使用指南

### 1️⃣ 载入图片
点击 **📁 选择图片** 上传，或点击 **📷 摄像头** 拍照。

### 2️⃣ 预处理（可选）
调整对比度和亮度，让种子与背景差异更明显。

### 3️⃣ 调整参数
| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 灰度阈值 | 低于该值的像素视为种子 | Otsu 自动 |
| 最小种子面积 | 滤除噪点 | 50 px² |
| 分离强度 | 控制粘连分离力度 | 中 (2) |
| 单颗种子半径 | 大约半颗种子的像素半径 | 6 px |

> 💡 点击 **🤖 自动** 按钮可自动计算最佳阈值（Otsu 法）

### 4️⃣ 选择区域
- **🔷 多边形**：点击顶点围出区域，靠近起点自动闭合
- **⬜ 矩形**：拖拽画出矩形框
- **🖼 全图**：直接分析整张图片

### 5️⃣ 执行计数
点击 **✨ 执行计数** 按钮（或按 `Enter` 键），等待结果。

### 6️⃣ 手动校正（可选）
开启 **✏️ 手动校正模式**：
- **点击绿色框** → 删除误检种子
- **点击暗色区域**（种子区域但未检出的部位） → 添加遗漏种子
- 支持撤销上次校正操作

### 7️⃣ 查看结果 & 发芽率
结果显示种子数量及面积统计。  
在 **🌱 总播种数** 输入框填入总数，自动计算 **发芽率**。

## 🔬 技术原理

1. **图像预处理**：对比度/亮度调整
2. **二值化**：灰度阈值分割（支持 Otsu 自动阈值）
3. **形态学填洞**：闭运算消除噪声孔洞
4. **距离变换**：Chamfer 3-4 近似欧氏距离图
5. **局部极大值搜索**：在指定半径内查找种子中心点
6. **分水岭分配**：基于种子的 BFS 区域生长，分离粘连颗粒
7. **连通域分析**：当分离关闭（强度=0）时直接使用连通域计数
