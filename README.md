# 🌱 专业版 · 小白菜种子自动计数仪 v2.1

> 基于距离变换 + 分水岭算法的高精度种子计数 Web 应用  
> 支持粘连分离、手动校正、实时预览 — 纯前端，无需后端

---

## ✨ 功能特性

- **🌾 自动计数**：上传图片或拍摄，自动检测并计数小白菜种子
- **🧠 智能粘连分离**：采用距离变换 + 分水岭算法，精准分离粘连种子
- **🔧 可调参数**：灰度阈值、最小面积、分离强度、种子半径均可实时调节
- **🤖 自动阈值 (Otsu)**：一键计算最佳二值化阈值
- **👁 二值化/距离图预览**：可视化图像处理中间结果，便于调参
- **✏️ 手动校正**：添加遗漏种子、删除误检，计数更可靠
- **📏 多种选区模式**：多边形、矩形、全图自由选择计数区域
- **📸 摄像头支持**：直接拍摄照片进行计数
- **⌨️ 全键盘快捷操作**：Enter 计数、Backspace 撤销点、C 校正、P 预览等
- **🌈 图像预处理**：对比度/亮度调节，适应不同拍摄环境
- **📊 统计信息**：显示面积均值/最小/最大，自动计算发芽率
- **📱 移动端适配**：响应式设计，支持触屏操作、双指缩放
